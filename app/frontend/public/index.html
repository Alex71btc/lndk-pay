⚙️ BOLT12 Pay Setup

Grundkonfiguration für Public URL, Lightning-Adressen und DNS-Modus.
⚠️ Hinweis für BOLT12 auf Umbrel
Damit BOLT12 Offers funktionieren, muss Onion Messaging in LND aktiviert sein.
Öffne diese Datei auf deinem Umbrel: 
/home/umbrel/umbrel/app-data/lightning/data/lnd/lnd.conf
Füge am Ende der Datei folgende Zeilen hinzu: 
[protocol]
custom-message=513
custom-nodeann=39
custom-init=39
Danach die Lightning Node neu starten.
🔐 Remote access security
If you expose this app to the internet, protect admin routes with Cloudflare Tunnel + Access or an equivalent access-control layer.
Recommended: protect /pay and /pay-login, while keeping required public payment endpoints reachable.

Manual DNS mode
Für BIP353 musst du bei deinem DNS-Provider einen TXT-Record setzen.
Schema: name.user._bitcoin-payment.domain.tld
Beispiel: bolt12.user._bitcoin-payment.pay.example.com
Inhalt: "bitcoin:?lno=<offer>"
Das konkrete Offer wird später von der App erzeugt. Cloudflare-Automation kann später optional aktiviert werden.
Cloudflare bleibt optional. Ohne Cloudflare funktioniert die App weiter, nur BIP353-DNS wird dann nicht automatisch gesetzt.
BOLT12 Pay Server

⚡ BOLT12 Offer & Alias Console

BOLT12 / BIP353 ist der primäre Zahlungsweg. LNURL und BOLT11 sind reine Fallbacks.
QR: BOLT12 Adresse
Lade QR...
Lade Adresse...
lnurl@yourdomain.com
Lade Offer...
Standardmäßig zeigt der QR die BOLT12-Adresse bolt12@example.com. Wenn du auf LNURL-Fallback umschaltest, wird stattdessen der LNURL-Inhalt angezeigt.
Diese Auswahl setzt den Mindestbetrag für ein neues Offer.
Erzeugt ein neues BOLT12-Offer über dein Backend und schaltet den QR automatisch auf das neue Offer.
BOLT11 Kompatibilitätsmodus
Normale Lightning-Rechnung für maximale Wallet- und WebLN-Kompatibilität.
Noch keine Rechnung erzeugt.
Dieser Modus erzeugt eine normale lnbc...-Rechnung über LND REST.
Alias Manager
Lightning-Adressen anlegen, veröffentlichen und verwalten.
Veröffentlichen erzeugt ein neues BOLT12-Offer und setzt bei Cloudflare automatisch den passenden BIP353 TXT-Record.
